StringExtensions.TitleCase Method

Creates a copy of this string that is titled cased. This follows the algorithm that both the first and last words of the string are capitalized, as are most other words in the string, with the exception of a list of excluded words (and, of, etc.) This list is a comma delimited assembly resource.
