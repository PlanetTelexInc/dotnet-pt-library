RegExPattern.NOT_ALPHA_NUMERIC_FOR_URL Field

[^0-9a-zA-Z\\s-\xC0-\xFF] - This does include whitespaces and hyphen characters. It also matches special extension ascii to include characters such as ê, ç, ñ. This regex expression was created specifically for a URL Rewriter.

public const string NOT_ALPHA_NUMERIC_FOR_URL = "[^0-9a-zA-Z\\s-À-ÿ]";
