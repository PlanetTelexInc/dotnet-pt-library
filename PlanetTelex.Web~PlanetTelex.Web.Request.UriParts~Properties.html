Public Instance Properties

Domain Gets the domain of the current request. If localhost, this value will include the virtual directory.
FirstDirectory Gets the first node that can be considered a directory (is not "localhost").

IsLocalhost Gets a value indicating whether this URI's domain is localhost.
LastDirectory Gets the last node that can be considered a directory (is not a page).
LastDirectoryNodeIndex Gets the index of the last node that can be considered a directory (is not a page).
Nodes Gets all the nodes (delimited by the '/' character) in the path.
Page Gets the requested page including its file extension.
PageExtension Gets the requested page file extension.
PageWithoutExtension Gets the requested page without its file extension.
Parameters Gets a name/value dictionary of parameters from the query part.
QueryPart Gets the query string of the request.
RelativeRoot Gets the correct relative root url element, taking into account if this is running under "localhost".
Scheme Gets the URI scheme.
SecondDirectory Gets the second node that can be concidered a directory (is not "localhost").
UriPath Get the path part of the URI, which includes the page. No domain or query part.
UriPathWithoutPage Get the path part of the URI without the page. No domain or or query part.
UriWithoutScheme Gets the URI without the scheme part.
UriWithoutSchemeAndQuery Gets the URI without scheme and query parts.
