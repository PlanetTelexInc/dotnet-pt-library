IRssItem Properties

The properties of the IRssItem interface are listed below. For a complete list of IRssItem interface members, see the IRssItem Members topic.

Public Instance Properties

RssItemAuthor Gets the email address of the author of this rss item. Example: author@site.com
RssItemComments Gets the URL of a page containing comments about this item. Example: http://www.myblog.org/cgi-local/mt/mt-comments.cgi?entry_id=290
RssItemDescription Gets the description of this rss item. Example: Some of the most heated chatter at the Venice Film Festival this week was about the way that the arrival of the stars at the Palazzo del Cinema was being staged.
RssItemLink Gets the link of this rss item. Example: http://nytimes.com/2004/12/07FEST.html
RssItemPublishedDate Gets the datetime when this item was published. Example(coverted to string): Sun, 19 May 2002 15:21:36 GMT
RssItemTitle Gets the title of this rss item. Example: Venice Film Festival Tries to Quit Sinking
